Practical Wireframing - a Workshop for Beginners

A good interface is key to any successful digital product. A good interface must be usable, understandable, and fit for purpose. Your users must be able to pick it up and use it right away, and have a great time while using your product.

Wireframing is a key step in the design process for any interface. In this 3h workshop you will design two interfaces from scratch. You can bring your own idea of what you want to design for one of the apps, or you can pick one from our hat of app ideas.

What you'll learn

What makes a good interface?

What does the UX design process look like?

User research methods

  • Focused brainstorming
  • Ethnography
  • Task analysis

Wireframing

  • Guided sketching for beginners
  • Where to find reference UI elements
  • The importance of iteration

What you’ll make

Lots and lots of wireframes! Over the course of the afternoon we’re going to run through a full design process for two user interfaces. The first interface will be a constructive task given by us, but the second interface can be for anything you like! If you have an idea for an app, web app, service, or any other type of software, this is a good opportunity to work on it while learning.

Who’s this class for

This class is designed for people working in the tech industry, and is focused on designing interfaces for software and websites. So if you work producing any sort of tech product, such as mobile / tablet apps, desktop software, web based software, this is the workshop for you.

If you’re a:

  • Manager - Understand the design process so you can effectively utilise designers within your team.
  • Developer - You might have built a few interfaces before, now learn how to design them.
  • Startup - New company can’t afford to lose customers just due to poor digital experience. Learn how to make experiences your customers will love!
  • Enterprise / Government - great interfaces allow software to scale to large sizes without putting strain on support staff. Learn how to make simple and effective interfaces for scale.

Who is teaching this class?

Eliot Slevin, a designer and developer who spends most of his time designing and building interfaces for the web, and is currently studying for a master's in media design.
